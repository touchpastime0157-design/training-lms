# システム開発見積分析報告書

本プロジェクト（Transport LMS）と同等のシステムを、AIを使用せずに一般的な受託開発会社に依頼した場合の想定コストと工数を算出しました。

## 1. 開発規模の評価
本システムには以下の高度な機能が含まれています。
*   **デスクトップアプリ化環境**: Electronを用いたWindowsアプリ基盤
*   **学習管理ロジック**: 12項目の進捗追跡、秒単位の視聴記録、月次リセット、複数人同時受講マージ
*   **管理機能**: ユーザー管理、パスワード認証、詳細な個別進捗ビュー
*   **帳票出力**: JavaScriptによる高度なPDFレポート生成（文字コード・レイアウト制御）
*   **高度なUI/UX**: モダンなデザイン、フルスクリーン制御、レスポンシブ対応

## 2. 想定工数（AIなしの場合）

標準的な開発チーム（PM 1名、エンジニア 1名、デザイナー 1名（部分投入））を想定。

| 工程 | 期間 | 主な作業内容 |
| :--- | :---: | :--- |
| **要件定義・画面設計** | 2週間 | 要件の言語化、ワイヤーフレーム作成、DB設計 |
| **デザイン作成** | 2週間 | モダンなUIデザイン、アイコン等のアセット制作 |
| **フロントエンド実装** | 4週間 | ダッシュボード、プレイヤー、複数人ログイン機能 |
| **管理機能・レポート実装** | 2週間 | 管理画面、PDF出力エンジン、個別進捗詳細ビュー |
| **テスト・デバッグ・修正** | 2週間 | Windows環境テスト、データ整合性テスト、バグ修正 |
| **納品・マニュアル作成** | 1週間 | マニュアル執筆、納品物(exe)の最終調整 |
| **合計** | **13週間** | **（約3ヶ月〜3.5ヶ月）** |

## 3. 想定費用（AIなしの場合）

日本国内の標準的な人月単価（80万円〜100万円/人月）で算出。

| 項目 | 金額（概算） | 備考 |
| :--- | :--- | :--- |
| **開発労務費** | 3,000,000円 | 技術者2名 × 1.5ヶ月分相当 |
| **プロジェクト管理費** | 300,000円 | 全体の10%程度 |
| **デザイン・UX設計費** | 500,000円 | オリジナルデザイン制作 |
| **諸経費・ライセンス等** | 200,000円 | 開発環境整備、動作検証用端末 |
| **合計見積金額** | **4,000,000円** | **(消費税別)** |

---

## 4. AI活用による効果の比較

| 比較項目 | AI活用（本件） | AIなし（従来手法） |
| :--- | :--- | :--- |
| **開発期間** | 数日間〜1週間程度 | 約3ヶ月 |
| **開発コスト** | 運用実費程度 | 約400万円 |
| **要件変更への柔軟性** | 即時反映 | 変更の都度、見積とスケジュール再調整 |

### 結論
今回の開発を外注した場合、**約400万円の費用と3ヶ月以上の納期**が必要になる規模のプロジェクトと言えます。AI（Antigravity）を活用することで、コストを極限まで抑えつつ、従来の10倍以上のスピードでリリースを実現したことになります。
