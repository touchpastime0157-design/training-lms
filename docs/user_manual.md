# 運行管理 LMS：受講生マニュアル

本システムは、トラック等の運行管理に関する知識を動画で学ぶための学習管理システムです。パソコン上で簡単に学習を進めることができます。

## 1. ログイン方法

アプリを起動すると、ログイン画面が表示されます。

1.  **受講者ログイン**タブを選択します。
2.  画面に表示されている名前の中から、**自分の名前**にチェックを入れます。
3.  複数人で一緒に動画を見る場合は、**一緒に見る人全員**にチェックを入れます。
4.  「受講を開始」ボタンを押して進みます。

## 2. ダッシュボード（メニュー画面）

ログイン後に表示される、学習の状況を確認する画面です。

*   **12の項目**: 12種類の講義動画が並んでいます。
*   **星マーク（進行状況）**: 各項目の右下に進捗度（％）が表示されます。
*   **「学習する」ボタン**: 動画を見るために、見たい項目の「学習する」ボタンを押します。

## 3. 学習の進め方（動画視聴）

動画画面が開いたら、再生ボタンを押して視聴を開始します。

*   **一時停止**: 動画内のボタンで一時停止できます。
*   **最大化**: 動画右下のボタンを押すと、画面いっぱいに大きく表示されます。
*   **中断（戻る）**: 左上の「＜」ボタンまたは「ダッシュボードに戻る」ボタンを押すと、現在の視聴場所を保存してメニュー画面に戻ります。

## 4. 修了（受講完了）について

*   **自動記録**: 動画を最後まで視聴（99.9%以上）すると、自動的に「受講完了」として記録されます。
*   **月次のリセット**: このシステムは月に一度の受講を目的としているため、月が変わると進捗は0%に戻ります。

## 5. アプリの終了方法

ログイン画面または一番最初の画面の右上にある「電源マーク（アプリを終了）」ボタンを押すと、アプリを閉じることができます。
