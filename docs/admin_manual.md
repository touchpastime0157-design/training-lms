# 運行管理 LMS：管理者向けマニュアル

管理者は、受講生の登録や学習状況の確認、社内提出用のレポート出力を行うことができます。

## 1. 管理画面への入り方

1. ログイン画面で「管理者ログイン」タブを選択します。
2. 管理者名（初期設定では admin）とパスワードを入力してログインします。

## 2. 受講生の管理

「ユーザー管理」タブ（初期画面）で行います。

*   **新規登録**: 画面右上の入力欄に名前を入力し、「追加」ボタンを押すと新しい受講生を登録できます。
*   **名前の変更**: 登録済みの名前の横にある「編集（鉛筆マーク）」ボタンで変更できます。
*   **削除**: 受講生を削除する場合は「削除（ゴミ箱マーク）」ボタンを押します。※一度削除するとその人の過去の受講記録も消えるためご注意ください。

## 3. 進捗状況の確認

「ダッシュボード」タブで行います。

*   **一覧表示**: 今月の受講生全員の進捗率が一覧で表示されます。
*   **詳細の確認**: 各受講生の右側にある「詳細↗」ボタンを押すと、全12項目のうち「どの動画を」「何パーセント」「いつ受講したか」を詳しく確認できます。
*   **過去の確認**: 画面上部の年月を選択することで、過去の受講記録も確認可能です。

## 4. レポート（PDF出力）

「レポート出力」タブで行います。

1. 出力したい「年」と「月」を選択します。
2. 画面右上の「PDFで出力」ボタンを押します。
3. 全員の名前と12項目の完了状況（○/×）がまとまった表がパソコンに保存されます。会議や監査の提出用としてご活用ください。

## 5. 管理者情報の変更

「管理者設定」タブで行います。
* 管理者の名前や、ログインに必要なパスワードを変更することができます。セキュリティのため、定期的な変更を推奨します。
